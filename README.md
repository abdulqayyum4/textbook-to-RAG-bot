# AI-Native Textbook

This is an educational resource for robotics and AI learners, focusing on Vision-Language-Action (VLA) systems that enable natural human-robot interaction.

## Project Structure

- `/docs/` - Contains all educational content in Markdown format
- `/docs/vla/` - Vision-Language-Action module content
- `/src/` - Docusaurus source files
- `/static/` - Static assets

## Available Content

The textbook currently includes the Vision-Language-Action module with chapters covering:
- LLM-Robot Convergence and voice-to-action processing
- Cognitive Planning for ROS 2 action sequences
- Complete VLA system integration
- Conceptual examples and diagrams

## Running the Development Server

```bash
npm start
```

The site will be available at [http://localhost:3000](http://localhost:3000)

## Technologies Used

- Docusaurus v3.1.0
- React
- Markdown/MDX
- Node.js
